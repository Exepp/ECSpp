EntitySpawner &  originSpawner,

Changes the archetype of the given entity.

Components that are not present in this spawner's archetype are destroyed

Components that are not present in originSpawner's archetype but are present in this spawner's archetype are allocated and can be constructed for the first time with Creator

Parameters
entA valid entity
entListList of entities to change the location data of ent
originSpawnerany OTHER spawner that owns entity
fnA callable object that can use the Creator instance to construct new components
Returns
True when changed the archetype, false otherwise (newArchetype was the same as ent's current archetype)
Exceptions
(Debugonly) Throws the AssertionFailed exception if ent is invalid or &originSpawner == this or if originSpawner is not the origin spawner of ent
