Changes the archetype of ent, removing components from toRemove list and adding the ones from toAdd list.

There are no overloaded versions of this function (for iterators) on purpose - it is more optimal to create a separate archetype for those situations

Parameters
entA valid entity
toAddComponents that will be added to ent (or kept it these are already there). toAdd has a priority over toRemove
toRemoveComponents that will be removed from ent (except for the ones specified in toAdd)
fnA callable object that can use the Creator instance to construct the added components
