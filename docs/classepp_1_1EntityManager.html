)
inline

Changes the archetype of ent, removing components from toRemove list and adding the ones from toAdd list.

There are intentionally no overloaded versions of this function (for iterators) - it is more optimal to create a separate archetype for those situations

Template Parameters
FnTypeCallable type that takes r-value reference to the EntityCreator
Parameters
entA valid entity
toAddComponents that will be added to ent (or kept it these are already there). toAdd has a priority over toRemove
toRemoveComponents that will be removed from ent (except for the ones specified in toAdd)
fnA Callable type that can use the Creator instance to construct the added components
Returns
Instance of IterTimeChange enum - ArchetypeCurrent when changed the archetype, ChangeFailed otherwise (newArchetype was the same as ent's current archetype)

Updates selection so that forEach member function can reach entities of the archetypes that were not present in its last update.
