Detailed Description

template<typename... CTypes>
class epp::Selection< CTypes >

A query to the EntityManager for entites that own a certain set of components.

Using forEach member function you can iterate over each entity that ows at least all of the components of CTypes... types. This class stores pointers to CPools and EntityPools of the EntitySpawners with archetypes that matches the specified requirements (CTypes and unwanted mask)

Template Parameters
CTypesA pack of component types used to select wanted entities
