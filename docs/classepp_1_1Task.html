Detailed Description

template<typename FnType, typename StepFnType, typename... CTypes>
class epp::Task< FnType, StepFnType, CTypes >

A simple tool for creating basic concurrent code.

This class eliminates race conditions by ensuring a one order of iteration for every task and keeping subtask always behind the uptasks. This way there are no concurrent reads or writes as long as EntityManager or other external object(pointer) is not used.
