Detailed Description

template<typename... CTypes>
template<bool IsConst>
class epp::Selection< CTypes >::Iterator< IsConst >

Iterator of the Selection.

Selection's iterator is invalid if it points to a location that would be impossible to get to with the current state of an EntityManager and only incrementation, i.e. changing the state of an EntityManager may invalidate iterators.

Here are the operations that may invalidates the iterators:

  • EntityManager::changeArchetype (has the overload for iterators that returns a next valid one)
  • EntityManager::destroy (has the overload for iterators that returns a next valid one)
  • EntityManager::clear
Template Parameters
IsConstA boolean that specifies whether this iterator operates on const Selection, or non-const

Jumps to an entity located at {entCell.spawnerId, entCell.poolIdx} or beyond.

Makes this iterator point to an entity located at {entCell.spawnerId, entCell.poolIdx} or if that location is not covered by the selection that this iterator comes from, to a first entity that lies beyond and IS covered by the selection or to the end if there is no such an entity.

Note
Iterator can only jump forward. Passing a data of an entity that lies before this iterator will make it jump to the end
Parameters
entCellData of an entity to jump to (or beyond)
